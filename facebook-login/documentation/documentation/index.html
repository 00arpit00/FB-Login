API for Facebook login integration.(made in LARAVEL framework.)

SOFTWARE REQUIREMENTS.

1. XAMPP ( for APACHE and mysql).

2. php.

3. COMPOSER.

 

Preparing Database.

1. First you have to create Database manually named as "fbuser".

2. Extract "fblogin.zip".

3. then locate .env file inside ../fblogin/facebook-login (folder).

4. open this with an php editor (or any editor).

5. find

DB_HOST=127.0.0.1
 
DB_PORT=3306

DB_DATABASE=fbuser

DB_USERNAME=root

DB_PASSWORD=
 
6. edit them with your configurations. 
 
7. then goto ../fblogin/facebook-login and open command window here.
 
8. then type " php artisan migrate " in command prompt to migrate all tables to the database.
 
9. now you are ready to access the API.
 
NOTE :-  IF YOU ARE HAVING PROBLEM WHILE MIGRATION, THEN IMPORT THE DATABASE FILE "fbuser.sql" to "fbuser" database.

GETTING STARTED.

1.Once you have done with the database, you are ready for the main process.

2. Goto directory ../fblogin/facebook-login and open Command Prompt there.

3. Type " php artisan serve " in the command prompt. 

4. Now open http://localhost:8000 for accessing the APP.

5. In these user need to click "Login With Facebook" button.

6. After that he/she will be redirected to facebook with a pop up of login-demo app(created by me) for accept or reject of request.

7. After accepting he would be redirected to the user's profile on the website with their basic details, like email address, profile picture, gender, etc.

8. DONE.!!
